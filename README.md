# Superstore Sales Analysis 📊

An end-to-end data science project analysing 5,009 sales orders
across 4 regions from 2014–2018.

## Dataset link for Download
-kaggle.com/datasets/vivek468/superstore-dataset-final

## 📊 Dataset
- Source: Kaggle 

## Tools Used
- Python, pandas, numpy
- matplotlib, seaborn
- Plotly (interactive dashboard)

## Key Insights
- West region leads in sales ($725K)
- Tables & Bookcases are loss-making sub-categories
- Discounts above 30% almost always result in losses
- Strong seasonal spike every Q4 (Nov–Dec)
